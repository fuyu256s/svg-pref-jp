# svg map of Japan prefectures
## info
- Data from  
  [Natural Earth](https://www.naturalearthdata.com/) > 
[1:10m Cultural Vectors](https://www.naturalearthdata.com/downloads/10m-cultural-vectors/) > ne_10m_admin_1_states_provinces.
- SVG files are created with [mapshaper](https://github.com/mbloch/mapshaper).
- UTM projections are used.
  - UTM zone 54 (EPSG:6691):  
  hokkaido, aomori, iwate, miyagi, akita, yamagata, fukushima, ibaraki, tochigi, gunma, saitama, chiba, tokyo, kanagawa, niigata, yamanashi, nagano, shizuoka
  - UTM zone 53 (EPSG:6690):  
  toyama, ishikawa, fukui, gifu, aichi, mie, shiga, kyoto, osaka, hyogo, nara, wakayama, tottori, shimane, okayama, hiroshima, tokushima, kagawa, ehime, kochi
  - UTM zone 52 (EPSG:6689):  
  yamaguchi, fukuoka, saga, nagasaki, kumamoto, oita, miyazaki, kagoshima, okinawa
- Version for now (I may edit them in the future).
